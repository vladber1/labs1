Пример клиент-серверного приложения для шифрования сообщений с помощью алгоритма Диффи-Хеллмана. 
Обмен ключами происходит в начале соединения, затем клиент и сервер могут отправлять друг другу зашифрованные сообщения, 
используя полученный общий ключ для шифрования и дешифрования. 

Программа работает в командной строке и управляется через ввод с клавиатуры.
