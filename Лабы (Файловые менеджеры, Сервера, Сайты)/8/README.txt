Этот код реализует простую регистрацию пользователей через веб-страницу. 
Пользователь может ввести свой логин и пароль, сервер хеширует пароль с помощью алгоритма SHA-256, 
добавляет пользователя в список пользователей и сохраняет список пользователей в файл users.json.
При запуске сервера на порту 5000 с помощью команды "python filename.py" сервер начинает слушать запросы на адресе 0.0.0.0. 
При запросе GET сервер возвращает форму для регистрации, при запросе POST сервер обрабатывает данные формы: 
проверяет поля логина и пароля на наличие, проверяет, что пользователь еще не зарегистрирован, 
хеширует пароль и добавляет пользователя в список пользователей.
